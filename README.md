# DI Validation Tool

A lightweight, responsive web tool for validating CSV data with clear error reporting and modular rule logic.

---

## 🚀 Features
- **CSV Validation**: Trade‑specific rules with batch processing
- **Secure Processing**: All validation runs locally in the browser — no external data transfer
- **Uniform Error Reporting**: Consistent format for seamless integration

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, Bootstrap
- **Logic**: JavaScript (modular validation pipeline)
